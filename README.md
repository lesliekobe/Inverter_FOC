# Inverter_FOC - 工业变频器FOC矢量控制

**完整纯C实现 / 工业级 / STM32F4 / DSP28335 双平台**

---

## 产品定位

低压三相矢量变频器（0.75kW ~ 75kW），主控芯片 STM32F4/F7/H7 或 DSP28335，纯C开发（无C++），适用于异步电机和永磁同步电机驱动。

## 核心功能

- ✅ V/F开环控制（异步电机，无需编码器）
- ✅ 矢量闭环FOC控制（异步电机/永磁同步电机）
- ✅ 过流/过压/欠压/过热/堵转/反时限过载保护
- ✅ 485 Modbus RTU通讯
- ✅ 多段速控制（8段）
- ✅ PID恒压/恒流闭环（供水/张力场景）
- ✅ 故障存储（32条循环记录）
- ✅ 参数掉电存储（Flash/EEPROM）
- ✅ 编码器采集（ABZ正交编码器）
- ✅ NTC温度检测 + 散热风扇智能控制

## 关键算法

### SVPWM 空间矢量调制

七段式对称SVPWM，谐波含量低，工业标配：
- 扇区判断 → 矢量时间计算 → 七段式插入
- 过调制处理（超出内切圆时等比例钳位）
- 死区补偿（STM32硬件BDTR寄存器）

### FOC 矢量控制（电流环 10kHz，速度环 1kHz）

```
三相电流 Ia,Ib → Clarke → Iα,Iβ
                        → Park  → Id,Iq
电流环PI(Id) → Vd
电流环PI(Iq) → Vq
                → 反Park → Vα,Vβ → SVPWM → 三相PWM
```

### 保护机制

- 瞬时过流：> I_max → 立即封锁PWM
- 反时限过载：I²t曲线，120%额定电流60s跳闸
- 堵转检测：转速=0 + 转矩电流>80%额定，2s跳闸

## Modbus寄存器映射（0x03读/0x06写）

| 地址 | 名称 | 说明 |
|------|------|------|
| 0x0000 | 运行命令 | 1=启动，0=停止 |
| 0x0001 | 频率给定 | 0.01Hz单位（如5000=50Hz）|
| 0x0002 | 电流反馈 | 0.01A |
| 0x0003 | 电压反馈 | 0.1V |
| 0x0004 | 转速反馈 | 1RPM |
| 0x0005 | 运行状态 | 0=停机,1=待机,2=运行... |
| 0x0006 | 故障代码 | 0=正常 |

## 调度时序

**裸机方案（低成本）：**
- 10kHz：PWM中断 → 电流环/采样
- 1ms：速度环、保护、APP逻辑
- 10ms：按键扫描、显示刷新
- 100ms：温度采集、参数存储

**FreeRTOS方案（大功率高性能）：**
- 任务优先级：MC(9) > APP(8) > COMM(6) > HMI(4) > MONITOR(2)
